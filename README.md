##  Project Overview
This project analyzes survey responses from data professionals to understand salary trends, job roles, skills, tools, and geographic distribution.

##  Tools Used
- Power BI
- Power Query (ETL)
- DAX (Basic Measures)
- Data Visualization

##  Dashboard Preview
![Dashboard Overview](https://github.com/GUREEZY/Data-Professional-Survey-PowerBi/blob/main/Data%20Professinal%20Survey%20-%20Dashboard.png)

##  Dataset
Survey data collected from data professionals across different roles, countries, and experience levels.

##  Data Preparation
- Cleaned raw survey data using Power Query
- Removed incomplete and invalid responses
- Standardized role, salary, and skill fields
- Created calculated columns and measures using DAX

##  Key Insights
- Average salary by job role
- Most common programming languages and tools
- Salary comparison across countries
- Work-life balance and job satisfaction trends

##  Dashboard Features
- Interactive slicers (Role, Country, Experience)
- KPI cards for salary and respondent count
- Dynamic charts and comparisons

## Outcome
Built an interactive Power BI dashboard providing clear insights into the data professional job market.
